# Hackspace Resources
Sample documentation, guidance, templates, etc


## Background
Organised by a number of trustees/directors of UK hackspaces, this repo contains reusable materials for starting or running a hackspace.  It complements the high level information available on the UK Hackspace Foundation website: http://http://www.hackspace.org.uk/
